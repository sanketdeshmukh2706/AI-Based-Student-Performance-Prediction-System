Welcome to the System

This platform uses to analyze student academic data, predict performance, and provide personalized feedback for improvement.

📊 Performance Analysis

Analyze subject-wise marks, attendance, and academic trends using interactive charts and dashboards.

🤖 ML Prediction

Predict student performance categories such as Excellent, Good, Average, or Poor using ML algorithms.

🧠 Personalized Feedback

Generate personalized academic feedback and suggestions to help students improve their performance.

About the System

The Student Performance Prediction System assists educational institutions in identifying students who require academic support at an early stage. It enables data-driven decision making for teachers and administrators.
