# 🎓 AI-Based Student Performance Prediction System

## 📌 Project Overview

The AI-Based Student Performance Prediction System is a machine learning-powered web application designed to predict student academic performance based on attendance, study habits, and subject-wise marks. The system helps educators and students identify performance trends and provides personalized feedback for improvement.

## 🚀 Features

* Student Performance Prediction using Machine Learning
* Attendance Analysis
* Subject-wise Marks Analysis
* Personalized Feedback Generation
* Interactive Data Visualizations
* Flask-based Web Application
* Performance Tracking and Monitoring
