# 🎓 Student Performance Prediction and Personalized Feedback System

## 📊 Project Overview
This project is a web-based student performance analysis system developed using Flask and Python. The application helps analyze student academic performance, generate report cards, visualize subject-wise marks, and provide personalized feedback based on student results.

The system includes an interactive dashboard where students and administrators can monitor academic progress and performance reports efficiently.

---

## 🎯 Objectives
- Analyze student academic performance
- Generate subject-wise performance reports
- Provide personalized feedback based on marks
- Visualize student performance using charts and graphs
- Create a user-friendly academic management system
